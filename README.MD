# SDL3-Bros

A small side-scroller game project in C++ with SDL3.

Right now this project is focused on learning and building the core systems step by step:

- movement
- gravity
- collision
- world space vs screen space
- camera and level foundations

This is an in-progress practice project, not a finished game.

To Clone:
```git clone --recursive https://github.com/alihanbays/SDL3-Bros```

To Run:
```cmake -B build ```, ```cmake --build build```, ```./build/SDL3-Bros```
